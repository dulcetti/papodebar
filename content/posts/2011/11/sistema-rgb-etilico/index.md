Quem mexe com tecnologia sabe o que é o sistema RGB, galera toda que trabalha com internet, a parte de web, principalmente desenvolvimento e webdesign. Talvez até você, nobre modafoca, que não sabe nada de códigos, de repente já se deparou com esse camarada chamado Sistema RGB.

<!--more-->

## Sistema RGB é muito melhor assim!

\[caption id="attachment_7717" align="aligncenter" width="600" caption="Isso ae pourra!"\]![Garrafas de Red Label, Green Label e Blue Label mostrando o que é o sistema RGB](images/Sistema-RGB.jpg "Garrafas de Red Label, Green Label e Blue Label mostrando o que é o sistema RGB")\[/caption\]

Isso sim é um sistema RGB etílico e de qualidades. Concordo em número e grau nessa sequência, primeiro o [Red Label, que não acho nada demais, bem fraco, o Green Label, 15 anos respeitável e claro, o nobre Blue Label](http://www.papodebar.com/os-scotchs-de-um-connoisseur/), respeitadíssimo.

Pra quem não conhece, o sistema RGB é um sistema de cores para monitores, dispositivos eletrônicos, assim como TVS, dentre outros, composto pelas cores **R**ed (Vermelho), **G**reen (Verde) e **B**lue (Azul), onde as cores são pegas através dessas três cores primárias, juntando com a luz.

## Finalizando

Vamos confessar, bem melhor o sistema RGB dessa forma, não é verdade? :)
